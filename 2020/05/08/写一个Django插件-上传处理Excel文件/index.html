原因

最近在做一个小的个人项目，经常需要更新数据，数据来自三方平台，数据形式主要以xls文件为主，由平台自助导出，格式和抬头较为固定，受不了一个个再导入自己平台，于是想着做一个插件，上传自动处理。

Django admin 分析
